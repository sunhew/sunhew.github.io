#### __계산식__ <br/>
* __result__: or문이 끝난뒤 a 혹은 b의 값이 저장됩니다. 따라서 값은 7이 됩니다.<br/>
* __2진수로 계산하기__: 7은 0111, 4는 0100입니다. 자세한 식은 아래를 참고해주세요.<br/>

## __열두번째 문제의 값 구하기__ <br/>
아래의 문제에서는 콘솔로그의 출력값을 구해야합니다.<br/>

```javascript
 function solution(a, b, c) {
    let answer = "YES", max;
    let total = a + b + c;

    if (a > b) max = a;
    else max = b;
    if (c > max) max = c;
    if (total - max <= max) answer = "NO";

    return answer;
}

console.log(solution(53, 93, 107));
```
### __문제 풀이__
#### __solution a, b, c의 값__ <br/>
* a, b, c의 값은 아래 콘솔로그에서 선언되었습니다. a: 53, b: 93, c: 107입니다. <br/>

#### __비교문__ <br/>
* __첫번째 비교문__: a의 값이 b보다 클경우 max의 값이 a가 저장되지만 이 경우에는 53 &gt; 93이기에 else가 실행되어 max값은 93이 됩니다.<br/>
* __두번째 비교문__: c가 max값보다 클경우 max의 값은 c의 값으로 변경됩니다. 이 경우에는 조건문이 충족되기에 max값은 107이 됩니다.<br/>
* __세번째 비교문__: a + b+ c의 값인 253과 max값을 뺀 값이 max 값보다 작거나 같을때 "NO"를 출력합니다.<br/>

### __정답__
* total값인 253과 max값인 107을 빼면 146이 나오는데 이 경우 107보다 크기에 YES가 출력됩니다.

아래의 문제에서는 콘솔로그의 출력값을 구해야합니다.<br/>

```javascript
 function solution(a, b, c) {
    let answer;
    if (a < b) answer = a;
    else answer = b;
    if (c <= answer) answer = c;
    return answer;
}
console.log(solution(15, 12, 11));
```
### __문제 풀이__
#### __a, b, c의 값과 answer의 값__ <br/>
* a, b, c의 값은 아래 콘솔로그에서 선언되었습니다. a: 15, b: 12, c: 11입니다. 또한 answer의 경우에는 아래 비교문에 따라 값이 변경됩니다.<br/>

#### __비교문__ <br/>
* __첫번째 비교문__: a의 값보다 b가 클 경우 answer의 값은 a가 됩니다. 만약 아닐경우에는 else로 넘어가 b가 입력됩니다. 따라서 answer의 값은 12입니다.<br/>
* __두번째 비교문__: answer의 값이 c보다 크거나 같을 경우에는 answer의 값이 c의 값으로 변경됩니다. 따라서 12는 11보다 크기에 answer의 값은 11이 됩니다.<br/>
* __종료__: answer의 값을 출력합니다.<br/>

### __정답__
* 11

## __열네번째 문제의 값 구하기__ <br/>
아래의 문제에서는 콘솔로그의 출력값을 구해야합니다.<br/>

```javascript
function solution(day, arr) {
    let answer = 0;
    for (let x of arr) {
        if (x % 10 == day) answer++;
    }
    return answer;
}

arr = [25, 23, 11, 47, 53, 17, 33, 40];
console.log(solution(0, arr));
```
### __문제 풀이__
#### __solution__ <br/>
* solution의 매개변수는 day, arr이며 각각의 값은 콘솔로그에 선언되어있습니다.<br/>

#### __계산식__ <br/>
* __answer__:answer의 값은 0으로 선언되었습니다.<br/>
* __let x of arr__: x는 Element이므로 arr의 또다른 이름이 됩니다.<br/>
* __x % 10 == day__: x와 10을 나눈 값의 나머지가 day와 같을경우 answer의 값이 증가합니다.<br/>

## __열여섯번째 문제의 값 구하기__ <br/>
아래의 문제에서는 result의 값을 구해야합니다.<br/>

```javascript
let a = 6, b = 9, c = 3, result;
    result = ++a + b++ + ++c;

    console.log(result);
    console.log(a + b + c);
```
### __문제 풀이__
#### __num1, num2의 초기값__ <br/>
* a: 6, b: 9, c: 3 result의 값은 선언되지않았습니다. <br/>

#### __계산식__ <br/>
* __++의 위치__: ++가 앞에 있는 경우에는 조건식의 처음부터 값이 증가하고 ++가 뒤에 있는 경우에는 조건식이 한번 끝난후에 값이 증가됩니다.<br/>
* __result__: a와 b, c의 값을 더한 값이 result의 값이 됩니다. 따라서 7 + 9 + 3의 합인 20이 result의 값이 됩니다. <br/>
* __조건문__: result의 계산이 끝나 b의 값이 증가합니다. 따라서 b의 값은 10이 됩니다.<br/>

### __정답__

```javascript
20
21
```

## __열일곱번째 문제의 값 구하기__ <br/>
아래의 문제에서는 a와 b의 출력값을 구해야합니다.<br/>

```javascript
 let a, b = 10;

    for (let i = 0; i < 5; i++) {
        a = i;
        b -= a;
    }
    console.log(a, b)
```
### __문제 풀이__
#### __a와 b의 초기값__ <br/>
* a와 b의 초깃값은 10으로 선언되었습니다. <br/>

#### __조건식 반복문__ <br/>
* __조건식__: i는 0부터 시작되며 i의 값이 5보다 작아야 합니다. 그리고 조건이 충족될떄까지 조건문이 끝날 때마다 i의 값이 1씩 증가합니다.<br/>
* __{}__: 조건이 맞을때마다 안에 들어가서 실행합니다.<br/>
* __a의 값 변경__: a의 값은 i의 값으로 변경됩니다. 첫번쨰 실행에서는 i가 0부터 실행되기 떄문에 a의 값은 0이 됩니다.<br/>
* __b의 값 변경__: b의 값에 a의 값을 뺴야합니다. 첫번쨰 실행에서는 a가 0이기 때문에 b의 값은 변경되지 않습니다. <br/>
* __반복문__: i의 값이 5와 같아지기 전까지 반복되며 a의 값이 증가됨에 따라 b의 값은 감소되어 저장됩니다.

### __과정과 정답__

```javascript
a = 0; b = 10;
a = 1; b = 9;
a = 2; b = 7;
a = 3; b = 4;
a = 4; b = 0;

4 0
```

```javascript
let num = [1, 5, 1, 2, 7, 5];
    for (let i = 0; i < 6; i++) {

        }
```

## __스무번째 문제의 값 구하기__ <br/>
아래의 문제에서는 콘솔로그의 출력값을 구해야합니다.<br/>

```javascript
let data = [10, 6, 7, 9, 3];
let temp;

for (let i = 0; i < 4; i++) {
    for (let j = i + 1; j < 5; j++) {
        if (data[i] > data[j]) {
            temp = data[i];
            data[i] = data[j];
            data[j] = temp;
        }
    }
}

console.log(data)
```
### __문제 풀이__
#### __data, temp의 초깃값__ <br/>
*  data의 값은 10, 6, 7, 9, 3이며 배열로 선언되었고 temp의 값은 선언되지 않았습니다.<br/>
#### __버블 정렬 알고리즘__ <br/>
*  버블 정렬은 서로 인접한 두 개의 숫자를 비교해서 큰 숫자를 뒤로 보내는 방식으로 정렬하는 알고리즘입니다. <br/>

#### __계산식__ <br/>

##### __외부 for문__ <br/>

````javascript
for (let i = 0; i < 4; i++)
````
이 부분은 i는 0부터 시작해서 4보다 작은 3이 될때까지 실행하게 됩니다. 위의 코드를 예로들면 배열의 길이는 5지만 4번만 반복하게 됩니다. 이렇게 하는 이유는 마지막 요소가 자동으로 정리되기 때문입니다.<br/>

##### __내부 for문__ <br/>

````javascript
for (let j = i + 1; j < 5; j++)
````
이 부분은 j는 i의 값에 1을 추가한 1부터 시작되며 5보다 작은 4가 될때까지 반복하여 총 4번 반복하게 됩니다. 이렇게 하는 이유는 인덱스 i부터 시작하여 배열의 끝까지 비교하기 위함입니다.<br/>

##### __요소 비교__ <br/>

````javascript
if (data[i] > data[j])
````
이 부분은 data[i]와 data[j]의 요소를 비교합니다. 만약 data[i]가 더 크다면 두 요소를 교환하게 됩니다.<br/>

##### __temp의 값 변경__ <br/>

````javascript
temp = data[i]
````
이 부분은 data[i]의 값이 temp에 저장합니다.<br/>

##### __data[i] 값 변경__ <br/>

````javascript
data[i] = data[j]
````
이 부분은 현재 요소 data[i]에 data[j]의 값을 저장합니다.<br/>

##### __data[i] 값 변경__ <br/>

````javascript
data[j] = temp
````
이 부분은 다음 요소 data[j]에 이전에 저장했던 data[i]의 값을 저장합니다.<br/>

### __정답__
