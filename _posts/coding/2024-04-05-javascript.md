## 블로그의 첫 글은 간단하게!!

```
{
        let a = 10;
        let b = 5;
        
        console.log(a + b); // 15 (더하기)
        console.log(a - b); // 5  (빼기)
        console.log(a * b); // 50 (곱하기)
        console.log(a / b); // 2  (나누기)
        console.log(a % b); // 0  (나머지)
}
```
